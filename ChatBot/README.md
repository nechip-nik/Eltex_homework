# Chatbot January 2025

Клиент-серверное приложение, реализующее чат-бота, который предоставляет информацию о фильмах, цитаты и шутки.
Клиент использует ncurses для текстового интерфейса, а сервер обрабатывает запросы клиентов, отправляя соответствующие данные.

## Возможности

### Клиент

- Предоставляет графический интерфейс для отправки сообщений;
- Позволяет пользователям регистрироваться и авторизоваться;
- Отправляет запросы на сервер и отображает полученные ответы.

### Сервер

- Обрабатывает сообщения, поступающие от клиентов;
- Поддерживает следующий набор команд:
	'/help'												# посмотреть список доступных команд
	'/info'												# посмотреть информацию о чат-боте
	'/exit'												# завершить сеанс с чат-ботом
	'/joke'			'funny/unfunny'						# выдача случайной шутки из категорий 'смешная/несмешная'
	'/quote'		'scientific/literary'				# выдача случайной цитаты из категорий 'научная/литературная'
	'/new_joke'		'funny/unfunny'			'content'	# добавить свою шутку
	'/new_quote'	'scientific/literary'	'content'	# добавить свою цитату
	'/name'			'title'								# поиск фильма по названию
	'/genre'		'genre'								# поиск фильма по жанру
	'/year'			'year'								# поиск фильма по году выпуска
	'/director'		'director'							# поиск фильма по режиссёру
- Хранит историю сообщений для каждого пользователя;
- Банит клиента по IP-адресу, который 3 раза ввёл неверную команду.

## Сборка

### Общий Makefile в корневой директории проекта

Цели:
> make all - полная сборка как клиентской, так и серверной частей проекта.
> make clean - очистка проекта от временных файлов и артефактов сборки.

После сборки проекта в корневой директории создаётся папка 'bin', содержащая исполняемые бинарные файлы для запуска клиента и сервера.

### Клиентский Makefile

Цели:
> make all - сборка клиентской части проекта.
> make clean - очистка временных файлов и артефактов сборки.

Сборка осуществляется с использованием библиотеки потоков и элементов интерфеса ('-lncurses', '-lpthread').

Если библиотека ncurses отсутствует в системе, может потребоваться установка следующих пакетов:
> sudo apt install libncurses5-dev libncursesw5-dev

После сборки проекта в клиентской директории создаётся папка 'obj', содержащая объектные модули '.c' файлов клиента.

### Серверный Makefile

Цели:
> make all - сборка серверной части проекта.
> make clean - очистка временных файлов и артефактов сборки.

Сборка осуществляется с использованием библиотеки потоков ('-lpthread').

После сборки проекта в серверной директории создаётся папка 'obj', содержащая объектные модули '.c' файлов сервера.

### Тесты

Цели:
> make all - сборка исполняемого файла тестов.
> make clean - очистка временных файлов и артефактов сборки.

## Запуск

1. Соберите бинарный файл, выполнив нужную цель из Makefile.
2. Перейдите в папку 'bin', там появятся бинарные файлы для запуска.
3. Запустите нужный бинарный файл:
	'./<название файла>' - запуск бинарных файлов.

## Прототипы функций

### Клиент

- 'void run_client();'
    Запускает клиентское приложение и инициализирует необходимые компоненты.

- 'void init_ncurses();'
    Инициализирует библиотеку ncurses для работы с текстовым интерфейсом.

- 'void create_windows();'
    Создает окна для отображения сообщений и ввода данных.

- 'void display_messages();'
    Отображает полученные сообщения в соответствующем окне.

- 'void display_input();'
    Отображает окно для ввода сообщений пользователем.

- 'void handle_input();'
    Обрабатывает ввод пользователя и отправляет сообщения на сервер.

- 'void clean_up();'
    Освобождает ресурсы и завершает работу приложения.

- 'void *receive_messages(void *arg);'
    Потоковая функция для получения сообщений от сервера.

- 'void add_message(const char *username, const char *message);'
    Добавляет новое сообщение в список сообщений для отображения.

- 'int login_or_register_window();'
    Отображает окно для выбора между входом и регистрацией.

- 'void registration_window();'
    Отображает окно для регистрации нового пользователя.

- 'void login_window();'
    Отображает окно для входа существующего пользователя.

- 'int send_login(const char *username, const char *password);'
    Отправляет данные для входа на сервер и возвращает статус операции.

- 'void *receive_from_server(void *arg);'
    Потоковая функция для получения данных от сервера.

- 'int send_registration(const char *username, const char *password);'
    Отправляет данные для регистрации нового пользователя на сервер.

- 'void send_message_to_server(const char *message);'
    Отправляет сообщение на сервер.

- 'int connect_to_server();'
    Устанавливает соединение с сервером и возвращает статус подключения.

### Сервер

- 'int create_server_socket(int port);'
    Создает серверный сокет, привязывает его к указанному порту и возвращает дескриптор сокета.

- 'int accept_client(int server_socket, struct sockaddr_in *client_addr);'
    Принимает входящее соединение от клиента и заполняет структуру client_addr информацией о клиенте. Возвращает дескриптор сокета клиента.

- 'void send_message(int client_socket, const char *message);'
    Отправляет сообщение клиенту через указанный сокет.

- 'User ErrorCode registration_user(const char *name, const char *password);'
    Регистрирует нового пользователя с указанным именем и паролем. Возвращает код ошибки, если регистрация не удалась.

- 'User ErrorCode login_user(const char *name, const char *password);'
    Выполняет вход пользователя с указанным именем и паролем. Возвращает код ошибки, если вход не удался.

- 'void ban_ip(const char *ip);'
    Запрещает доступ к серверу для указанного IP-адреса.

- 'int is_ip_banned(const char *ip);'
    Проверяет, заблокирован ли указанный IP-адрес. Возвращает 1, если IP заблокирован, и 0 в противном случае.

- 'void send_help_info_file(char *filename, int socket, const char *file_path);'
    Отправляет файл с информацией о помощи клиенту через указанный сокет.

- 'void send_formatted_message(int socket, const char *text);'
    Отправляет отформатированное сообщение клиенту через указанный сокет.

- 'int write_to_file(const char *filename, const char *str);'
    Записывает строку в указанный файл. Возвращает 0 при успешной записи, иначе -1.

- 'void delete_user_history_files();'
    Удаляет файлы истории пользователей.

- 'void *handle_client(void *arg);'
    Обрабатывает взаимодействие с клиентом в отдельном потоке. Принимает указатель на структуру с данными клиента.

- 'int load_data(const char *filename, void **data, int *count, int type);'
    Загружает данные из указанного файла в память. Возвращает количество загруженных элементов.

- 'int parse_film(char *line, struct FILM *film);'
    Парсит строку, представляющую фильм, и заполняет структуру FILM. Возвращает 0 при успешном парсинге, иначе -1.

- 'int parse_joke_quote(char *line, struct JOKE_QUOTE *joke_quote);'
    Парсит строку, представляющую шутку или цитату, и заполняет структуру JOKE_QUOTE. Возвращает 0 при успешном парсинге, иначе -1.

- 'void free_films(struct FILM *films, int film_count);'
    Освобождает память, занятую массивом фильмов.

- 'void free_jokes_quotes(struct JOKE_QUOTE *jokes_quotes, int joke_quote_count);'
    Освобождает память, занятую массивом шуток и цитат.

- 'void free_info_help(char **info_help, int info_help_count);'
    Освобождает память, занятую массивом информации о помощи.

- 'void search_films(const char *username, int client_socket, const char *search_term, enum SearchType type);'
    Выполняет поиск фильмов по заданному термину и отправляет результаты клиенту.

- 'void search_quotes(const char *username, int client_socket, const char *search_type, int user_id, struct JOKE_QUOTE *user_quotes, int user_joke_count);'
    Выполняет поиск цитат по заданному типу и отправляет результаты клиенту.

- 'void search_jokes(const char *username, int client_socket, const char *search_type, int user_id, struct JOKE_QUOTE *user_jokes, int user_joke_count);'
    Выполняет поиск шуток по заданному типу и отправляет результаты клиенту.
